# Host Migration
Host migration for my bachelor's thesis (Best Student Thesis 2022-2025 - Highest Distinction)

Demo uses Mirror for networking (KCP transport) in Unity and works on Steam and LAN.
 
Researched and published a thesis on the feasability of host-migration in online co-op games with statistical analysis
